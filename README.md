# cyclistic-bike-share-analysis
Google Data Analytics Capstone Project - Cyclistic bike-share analysis using Python and pandas
## Overview
This project is part of the Google Data Analytics Professional Certificate capstone. 
It analyzes 13 months of Cyclistic bike-share trip data (May 2025 – May 2026) to 
identify behavioral differences between annual members and casual riders.

## Business Question
How do annual members and casual riders use Cyclistic bikes differently?

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
1. Members take nearly twice as many trips as casual riders, primarily on weekdays, suggesting commuting behavior.
2. Casual riders take longer trips on weekends, indicating recreational use.
3. Both user types prefer electric bikes over classic bikes.

## Recommendations
1. Offer introductory discounted memberships to encourage casual riders to adopt the service for daily commuting.
2. Partner with local businesses along service routes to offer membership-linked discounts for recreational riders.
3. Increase electric bike availability at high-demand stations, particularly during peak weekend hours.
